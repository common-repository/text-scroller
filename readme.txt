== Description ==

Set Scrolling Message for website


== Installation ==

1. Upload 'text-scroller' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the Text Scroller 
4. You can enter your scrolling message.
5. Click on "Submit" button.
